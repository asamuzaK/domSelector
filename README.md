Retrieve DOM node from the given CSS selector.
**Experimental**

## Install

```console
npm i @asamuzakjp/dom-selector
```

## Usage

```javascript
const {
  matches, closest, querySelector, querySelectorAll
} = require('@asamuzakjp/dom-selector');
```
